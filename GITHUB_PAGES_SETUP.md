# Host Your Birthday Wish on GitHub Pages

Follow these steps to get a shareable URL for WhatsApp:

---

## Step 1: Create a new repository on GitHub

1. Go to **https://github.com/new**
2. Set the repository name (e.g. `birthday-wish` or `happy-birthday`)
3. Choose **Public**
4. **Do NOT** add a README, .gitignore, or license (we already have these)
5. Click **Create repository**

---

## Step 2: Push your project

Copy the commands from your new repo page (they'll look like this, but use YOUR username and repo name):

```bash
cd "d:\wish project"
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

Replace `YOUR_USERNAME` with your GitHub username and `YOUR_REPO_NAME` with your repo name.

---

## Step 3: Enable GitHub Pages

1. In your repo, go to **Settings** → **Pages** (left sidebar)
2. Under **Source**, select **Deploy from a branch**
3. Under **Branch**, select **main** and **/ (root)**
4. Click **Save**
5. Wait 1–2 minutes for the site to deploy

---

## Step 4: Get your URL

Your site will be live at:

**https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/**

Example: If your username is `lenin` and repo is `birthday-wish`:
→ **https://lenin.github.io/birthday-wish/**

Share this URL on WhatsApp — anyone who clicks it will open your birthday surprise in their browser.
